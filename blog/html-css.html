Classes vs Ids

10 April 2023

Classes allow you to style multiple div objects at once. If multiple divs are very similar to each other then there is not point changing all the styling for each object individually, it's much easier to change it for one big group. Ids allow for indiviudal styling adjustment even for divs that are in a group. This is known as overriding a styling property.
